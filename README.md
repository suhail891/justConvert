# 🚀 ImageToPDF

A Full Stack Web Application to **Convert Images into PDF** and **Merge Multiple PDFs** into a single document.

This project allows users to upload images, convert them into PDF files, and merge multiple PDFs into one. Built using modern full‑stack technologies with clean architecture and scalable backend structure.

---

# 📌 Features

✅ Convert Images to PDF
✅ Merge Multiple PDFs
✅ Upload Multiple Files
✅ Fast File Processing
✅ Clean UI
✅ Download Generated PDF
✅ Error Handling

---

# 🛠️ Tech Stack

## Frontend

* React.js
* Axios
* HTML5
* CSS3

## Backend

* Node.js
* Express.js
* Multer (File Upload)
* PDF-lib (PDF Processing)

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```
git clone https://github.com/your-username/ImageToPdf.git
```

```
cd ImageToPdf
```

---

# 🔧 Backend Setup

Navigate to backend folder

```
cd backend
```

Install dependencies

```
npm install
```

Run backend server

```
npm start
```

Server running on:

```
http://localhost:5000
```

---

# 💻 Frontend Setup

Navigate to client folder

```
cd client
```

Install dependencies

```
npm install
```

Run frontend

```
npm start
```

Frontend running on:

```
http://localhost:3000
```

# 📤 API Endpoints

## Convert Image to PDF

```
POST /api/pdf/convert
```

Upload images and convert into PDF

---

## Merge PDFs

```
POST /api/pdf/merge
```

Upload multiple PDFs and merge into one
